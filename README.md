# MLB Research Vault

Agent-maintained knowledge base for MLB betting research, focused on fading the Colorado Rockies and Chicago White Sox.

## Structure

```
raw/           → Unprocessed source material (articles, data, images)
compiled/      → LLM-synthesized wiki (concepts, summaries, daily reports)
output/        → Rendered artifacts (charts, slides)
tools/         → Scripts and search utilities
```

## Principles

- This vault is agent-written. Human rarely edits directly.
- Q&A outputs get filed back into compiled/ to compound knowledge.
- Raw data preserves original sources; compiled layer adds synthesis.
- Only high-value artifacts get promoted to creed-thoughts.

## Topics

- Colorado Rockies analysis (pitching, batting, schedule, weather)
- Chicago White Sox analysis
- Fade strategy tracking (ML, RL, SGP)
- Starting pitcher matchups and ERA trends
- Betting market analysis and odds movement
